### 문제 설명

<p>문자열 S의 부분 문자열 중에서 팰린드롬인 것 중 가장 긴 것의 길이를 구하는 프로그램을 작성하시오.</p>

### 입력

### 출력 

 <p>가장 긴 팰린드롬 부분 문자열의 길이를 출력한다.</p>
